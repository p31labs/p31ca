## Design Principles

The P31 EDE follows three core principles:

**Delta Topology** — Reject hierarchy and centralization. Every subsystem forms a
self-supporting triangle of redundancy. The three-agent AI mesh (Claude, DeepSeek, Gemini)
mirrors this: no single model is a dependency.

**Local-First** — All data lives on the operator's machine first. Neo4j runs locally.
Ollama provides offline AI. The system degrades gracefully when cloud services are unavailable.

**Cognitive Sovereignty** — The operator's attention is the scarcest resource. Every design
decision optimizes for spoon conservation: progressive disclosure, voltage scoring, batched
notifications, and adaptive UI complexity.

## Protocol

All hardware communication uses the P31 serial protocol:

| Parameter | Value |
|-----------|-------|
| Magic byte | `0x31` (Phosphorus-31) |
| CRC8 polynomial | `0x31` (CRC8-MAXIM) |
| CRC8 init | `0xFF` |
| Frame encoding | COBS |
| Delimiter | `0x00` |
| Baud | 115200 |
| USB | Native CDC (GPIO19/20) |

Frame structure: `COBS_ENCODE(magic | cmd | payload | crc8) + 0x00`
