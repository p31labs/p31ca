## What is P31?

P31 Labs builds **open-source assistive technology** for neurodivergent individuals.
The Everything Development Environment (EDE) is our sovereign development ecosystem —
a drop-in devcontainer that orchestrates hardware, firmware, AI mesh networking,
and cognitive interface tools.
